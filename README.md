# 🌐 Responsive Landing Page

A modern and responsive landing page developed as **Task 01** of the **Prodigy InfoTech Web Development Internship**.

The project demonstrates a clean UI with a fixed navigation bar, smooth scrolling, hover effects, and responsive design using HTML, CSS, and JavaScript.

---

## 📌 About

This landing page includes:

- Responsive layout for all screen sizes
- Fixed navigation bar
- Smooth scrolling navigation
- Hover effects on menu items
- Dynamic navbar color on scroll
- Modern UI design
- Clean and organized code

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
